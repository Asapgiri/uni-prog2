# How to install NuGet in Visual Studio

## For tests:
1. Create a new `class library`
2. Right click -> Add NuGet packages
    - Download `NUnit`
    - Download `NUnit3TestAdapter`
    - Download `Microsoft.NET.Test.Sdk`

## For mocks:
1. Create a new `class library`
2. Right click -> Add NuGet packages
    - Download `Moq`
